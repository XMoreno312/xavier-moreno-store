# /beats assets

## Optional masthead video

Drop a file at:

```
public/beats/masthead.mp4
```

…and `components/beats/BeatsMasthead.jsx` will fade it in automatically over
the gradient plate (~45% opacity, screen blend). Keep it dark, slow, and
abstract — fabric, smoke, distant city lights, water surface. ~10s loops are
ideal. If no file is present, the gradient plate stands on its own.
